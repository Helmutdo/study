# AI Tutor — Python desde cero

Curso de Python en archivos `.py` comentados. Cada archivo es una lección que puedes **leer** y **ejecutar**:

```bash
python3 AI_tutor/00_introduccion.py
python3 AI_tutor/01_variables_tipos.py
# ... etc.
```

## Orden de lecciones

| # | Archivo | Contenido |
|---|---------|-----------|
| 00 | `00_introduccion.py` | Qué es Python, `print()`, comentarios, indentación |
| 01 | `01_variables_tipos.py` | Variables, `int`, `float`, `str`, `bool`, `None`, `type()` |
| 02 | `02_operadores.py` | Aritméticos, comparación, lógicos, `in`, `is` |
| 03 | `03_entrada_salida.py` | `input()`, `print()`, f-strings, formateo |
| 04 | `04_condicionales.py` | `if`, `elif`, `else`, truthy/falsy |
| 05 | `05_bucles.py` | `for`, `while`, `break`, `continue`, `range()` |
| 06 | `06_listas_tuplas.py` | Listas, tuplas, slicing, comprehensions |
| 07 | `07_diccionarios_conjuntos.py` | `dict`, `set`, operaciones |
| 08 | `08_funciones.py` | `def`, parámetros, `*args`, `**kwargs`, `lambda` |
| 09 | `09_manejo_archivos.py` | `open()`, `with`, leer/escribir archivos |
| 10 | `10_clases_objetos.py` | Clases, `__init__`, herencia, `super()` |
| 11 | `11_modulos_paquetes.py` | `import`, módulos, paquetes, `__name__` |
| 12 | `12_errores_excepciones.py` | `try`/`except`/`else`/`finally`, `raise` |
| 13 | `13_resumen.py` | Resumen y próximos pasos |

## Requisitos

- Python 3.8+ (recomendado 3.10+).
- Nada más: solo el intérprete estándar.

## Consejos

1. Lee el código y los comentarios de cada archivo.
2. Ejecuta cada lección y modifica cosas para ver qué pasa.
3. Sigue el orden 00 → 13; cada lección usa conceptos de las anteriores.
